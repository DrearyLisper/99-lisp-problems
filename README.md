# Solving 99 Lisp Problems in Clojure

Use ```lein repl``` to start a REPL with all solutions available.

## Problem 01

(*) Find the last box of a list.

``` clojure
main=> (p01 '(a b c d))
(d)
```

## Problem 02

(*) Find the last but one box of a list.

``` clojure
main=> (p02 '(a b c d))
(c d)
```
